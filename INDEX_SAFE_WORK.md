# Index: Safe Work Materials

## Основные файлы

- `SAFE_OSINT_WORKSPACE.md` — общий безопасный рабочий процесс.
- `prompts/safe-research-prompts.md` — промпты для исследования, OCR, отчета и обезличивания.
- `templates/research-log.md` — журнал исследования.
- `templates/general-report-template.md` — общий шаблон отчета.
- `templates/official-request-notes.md` — заметки для официального обращения.
- `checklists/publication-checklist.md` — проверка перед сохранением, отправкой или публикацией.

## Работа с изображениями

- `workflows/image-ocr-workflow.md` — извлечение видимого текста с изображения.
- `workflows/vehicle-image-safe-workflow.md` — безопасный анализ изображения транспорта.
- `guides/vehicle-image-notes.md` — формат заметок по изображению транспорта.
- `exercises/safe-visual-ocr-exercises.md` — учебные упражнения по OCR.

## Ресурсы

- `resources/visual-text-recognition-repositories.md` — публичные репозитории для изучения OCR-пайплайнов.

## Правило использования

Работать только с открытыми, разрешенными и учебными материалами. Перед публикацией скрывать лишние частные детали и фиксировать уровень уверенности.
