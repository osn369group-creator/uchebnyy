# План обучения OSINT

## Модуль 1. Основы OSINT

Цель: понять, что такое OSINT и чем он отличается от незаконного сбора данных.

Темы:

- открытые источники информации;
- факт, гипотеза, вывод;
- уровень уверенности;
- этика и правовые границы;
- документирование процесса.

Практика:

- взять публичную новость и выделить факты, предположения и неизвестные данные;
- найти 2–3 независимых подтверждения одного факта.

## Модуль 2. Поисковые операторы

Темы:

- точные фразы;
- исключение слов;
- поиск по сайту;
- поиск по типу файла;
- поиск по дате;
- сравнение результатов разных поисковых систем.

Практика:

- найти официальный документ на сайте организации;
- сравнить результаты обычного запроса и запроса с операторами.

## Модуль 3. Проверка источников

Темы:

- первичные и вторичные источники;
- репутация источника;
- дата публикации;
- конфликт интересов;
- признаки недостоверности.

Практика:

- сравнить официальный пресс-релиз, СМИ и пост в соцсети;
- оценить, какой источник сильнее и почему.

## Модуль 4. Изображения, карты и геоданные

Темы:

- обратный поиск изображений;
- метаданные без обхода приватности;
- публичные карты;
- проверка места по открытым визуальным признакам.

Практика:

- использовать учебное изображение или публичный пример;
- описать проверяемые признаки без идентификации частных лиц.

## Модуль 5. Технический OSINT

Темы:

- домены и DNS на базовом уровне;
- публичные сведения о сертификатах;
- открытые репозитории;
- публичная документация продуктов;
- безопасный анализ без сканирования чужих систем.

Практика:

- изучить публичный домен организации только через разрешённые справочники;
- составить карту публичных источников информации об организации.

## Модуль 6. Отчётность

Темы:

- структура отчёта;
- журнал поиска;
- ссылки и даты доступа;
- уровень уверенности;
- ограничения вывода.

Практика:

- подготовить короткий отчёт по безопасному учебному кейсу;
- отдельно указать, что не удалось подтвердить.

## Итоговый учебный проект

Выбрать безопасный объект исследования: публичную организацию, открытый проект, новостное событие или синтетический кейс. Подготовить отчёт с источниками, уровнем уверенности и ограничениями.
