# Учебный OSINT-репозиторий

Этот репозиторий предназначен для безопасного и законного изучения OSINT — работы с открытыми источниками информации.

## Цель

Создать базу знаний и практический набор правил для OSINT-агента, который помогает:

- формулировать исследовательские вопросы;
- искать информацию только в открытых и разрешённых источниках;
- проверять факты по нескольким независимым источникам;
- отделять подтверждённые факты от предположений;
- документировать процесс поиска и уровень уверенности;
- соблюдать этику, приватность и законодательство.

## Важные ограничения

Этот репозиторий не предназначен для доксинга, преследования, взлома, фишинга, обхода приватности, скрытого наблюдения, сбора паролей, кражи аккаунтов или публикации персональных данных.

Не добавляйте сюда реальные персональные данные, приватные сведения, токены, пароли, ключи API, дампы баз данных, закрытые документы или материалы, полученные незаконным путём.

## Как использовать

1. Сначала прочитать `ETHICS_AND_SAFETY.md`.
2. Затем изучить `AGENT_INSTRUCTIONS.md`.
3. Проходить темы из `OSINT_LEARNING_PLAN.md`.
4. Использовать `REPORT_TEMPLATE.md` и `notes/research-log-template.md` для документирования учебных исследований.
5. Выполнять только безопасные упражнения из папки `exercises/`.
