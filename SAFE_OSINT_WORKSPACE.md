# Safe OSINT Workspace

Этот файл описывает безопасную рабочую среду для учебного OSINT.

## Разрешено

- Работать только с открытыми и законно доступными источниками.
- Проверять факты по нескольким независимым источникам.
- Вести журнал исследования: цель, источник, дата доступа, вывод, уровень уверенности.
- Сохранять только минимально необходимую информацию.
- Удалять или скрывать частные идентификаторы перед публикацией материалов.
- Готовить нейтральные отчеты без обвинений и без раскрытия частной информации.

## Рабочий цикл

1. Сформулировать цель.
2. Проверить законность и необходимость сбора данных.
3. Найти открытые источники.
4. Зафиксировать ссылки и дату доступа.
5. Сравнить данные минимум из двух источников.
6. Отделить факты от предположений.
7. Указать уровень уверенности.
8. Обезличить материалы перед сохранением или публикацией.

## Формат вывода

- Краткое резюме.
- Источники.
- Проверенные факты.
- Неподтвержденные гипотезы.
- Ограничения исследования.
- Рекомендованные законные следующие шаги.
