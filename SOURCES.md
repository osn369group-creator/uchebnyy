# Категории источников для безопасного OSINT

Этот список описывает типы источников, которые можно использовать в учебных исследованиях.

## 1. Официальные источники

- сайты организаций;
- официальные пресс-релизы;
- государственные реестры и публикации;
- публичная документация;
- страницы поддержки и справки.

## 2. Новостные источники

- крупные СМИ;
- местные СМИ;
- отраслевые издания;
- фактчекинговые проекты.

Важно сравнивать несколько источников и учитывать дату публикации.

## 3. Архивы и кэши

- интернет-архивы;
- сохранённые публичные страницы;
- исторические версии документов.

Использовать только для анализа публично доступной информации.

## 4. Технические открытые источники

- открытые репозитории;
- публичная документация API;
- DNS-справочники;
- публичные сведения о сертификатах;
- открытые базы уязвимостей и CVE.

Не использовать для атак, сканирования чужих систем без разрешения или обхода доступа.

## 5. Карты и геоданные

- публичные карты;
- спутниковые снимки;
- справочники организаций;
- публичные транспортные и географические данные.

Не использовать для слежки за частными лицами.

## 6. Академические и исследовательские источники

- научные статьи;
- отчёты исследовательских центров;
- доклады общественных организаций;
- открытые датасеты.

## Правило проверки

Один источник — это повод для гипотезы. Несколько независимых надёжных источников — основа для уверенного вывода.
