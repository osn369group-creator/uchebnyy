# Индекс раздела Кинодвиж VK OSINT

## Основные файлы

- `README.md` — назначение и структура раздела.
- `AGENT_INSTRUCTIONS.md` — рабочие инструкции агента.
- `workflows/vk-public-monitoring-workflow.md` — процесс работы с открытыми VK-публикациями.
- `templates/vk-evidence-card.md` — карточка доказательства.
- `templates/vk-osint-report.md` — шаблон сводки.
- `checklists/vk-privacy-safety-checklist.md` — безопасность и приватность.
- `resources/vk-github-study-list.md` — список GitHub-репозиториев для изучения VK API и открытых публикаций.
- `prompts/vk-safe-search-prompts.md` — безопасные рабочие запросы.

## Как использовать

1. Открыть `AGENT_INSTRUCTIONS.md`.
2. Выбрать workflow под задачу.
3. Для каждого найденного поста заполнить `vk-evidence-card.md`.
4. Сформировать итог через `vk-osint-report.md`.
5. Проверить отчёт по `vk-privacy-safety-checklist.md`.

## Приоритет

Правила безопасности и приватности имеют приоритет над полнотой сбора данных.
