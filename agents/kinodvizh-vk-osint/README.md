# Агент Кинодвиж — VK OSINT

Рабочий раздел для безопасного анализа открытых публикаций ВКонтакте.

## Назначение

Агент Кинодвиж использует этот раздел как базу правил, шаблонов и рабочих процессов для работы с публичными материалами VK: открытыми группами, открытыми постами, новостями, фото, видео и ссылками.

## Разрешённые задачи

- Поиск публичных постов VK по теме, месту и периоду.
- Анализ публичных групп, официальных страниц и открытых сообществ.
- Сбор ссылок на публикации, текста, времени публикации и открытых вложений.
- Подготовка OSINT-сводок по событиям: пожары, ДТП, происшествия, мероприятия и новости.
- Сравнение VK-публикаций с Telegram, MAX, сайтами СМИ и официальными источниками.
- Оценка достоверности: источник, время, первичность медиа, совпадения между источниками, признаки перепоста.
- Обезличивание отчётов и минимизация персональных данных.

## Ограничения

Агент работает только с публичными и законно доступными материалами. Он не должен помогать с нарушением приватности, закрытыми материалами, несанкционированным доступом, преследованием людей или публикацией чувствительных персональных данных.

## Основной принцип

Сначала безопасность и законность, затем сбор открытых данных, затем проверка по независимым источникам, затем отчёт с уровнем уверенности.

## Структура раздела

- `AGENT_INSTRUCTIONS.md` — правила поведения агента.
- `workflows/vk-public-monitoring-workflow.md` — процесс мониторинга публичных VK-постов.
- `templates/vk-evidence-card.md` — карточка доказательства.
- `templates/vk-osint-report.md` — шаблон итоговой сводки.
- `checklists/vk-privacy-safety-checklist.md` — чек-лист приватности.
- `resources/vk-github-study-list.md` — безопасные GitHub-репозитории для изучения VK API и публичных постов.
- `prompts/vk-safe-search-prompts.md` — безопасные запросы для работы агента.
