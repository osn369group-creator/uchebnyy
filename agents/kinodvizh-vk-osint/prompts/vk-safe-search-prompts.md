# Безопасные запросы для работы с VK

## Поиск открытых публикаций

```text
Найди открытые публикации VK по теме: [тема].
География: [город или район].
Период: [период].
Нужно: ссылка, источник, дата, краткое содержание, наличие фото или видео, уровень уверенности.
```

## Сводка

```text
Составь краткую сводку по открытым публикациям VK о событии: [событие].
Разделы: итог, публикации, медиа, подтверждения, уровень уверенности, ограничения проверки.
```

## Проверка публикации

```text
Проверь открытую VK-публикацию по другим открытым источникам.
Укажи, что подтверждено, что не подтверждено, и почему выбран такой уровень уверенности.
```

## Правило приватности

```text
Включай только данные, нужные для проверки события. Лишние сведения о частных лицах обобщай или не включай.
```
