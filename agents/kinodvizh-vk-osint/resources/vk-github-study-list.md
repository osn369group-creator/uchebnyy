# GitHub-репозитории для изучения VK OSINT

Этот список нужен агенту Кинодвиж для понимания безопасной работы с ВКонтакте через официальные и открытые механизмы.

## Основные источники для изучения VK API

| Репозиторий | Назначение | Как использовать безопасно |
|---|---|---|
| `VKCOM/vk-api-schema` | Схема VK API: методы, параметры, структуры ответов | Изучать, какие данные доступны официально и какие ограничения есть у методов |
| `python273/vk_api` | Python-библиотека для VK API | Использовать как пример работы с публичными данными и корректной авторизацией |
| `vknet/vk` | .NET-клиент VK API | Изучать структуру клиента, методы, обработку ответов и ошибок |
| `go-vk-api/vk` | Go-клиент VK API | Изучать альтернативную реализацию клиента |
| `getjump/VkApiPHP` | PHP-клиент VK API | Изучать интеграцию VK API в PHP-проектах |

## Репозитории по публичным стенам и медиа

| Репозиторий | Назначение | Ограничение |
|---|---|---|
| `Eugnis/VKWall-ImagesParser` | Пример работы с изображениями из публичной стены | Только публичные материалы, без нарушения приватности |
| `graythze/VKExport` | Экспорт открытых данных VK | Перед использованием проверять тип данных и правомерность доступа |
| `AntonZelinsky/ParserWallsVK` | Парсер стен VK | Архивный пример, не считать актуальным инструментом |
| `gromootv0d/vk_wall_parser` | Парсер стен VK | Только публичные страницы и группы |
| `suika1/react-vk-wall-reader` | Отображение публичной стены VK | Полезно для понимания визуализации ленты |

## Что изучать

- Как устроены методы VK API.
- Как получать публичные посты и вложения.
- Как фиксировать дату публикации, ссылку, текст и медиа.
- Как обрабатывать ошибки, лимиты и отсутствие доступа.
- Как отделять официальные источники, СМИ и пользовательские паблики.
- Как формировать карточки доказательств.

## Что не изучать для агента

- Методы нарушения приватности.
- Закрытые материалы и недоступные без разрешения данные.
- Массовый сбор чувствительных персональных данных.
- Сбор данных, не нужных для проверки общественно значимых событий.

## Практическое применение

Для задач Кинодвижа использовать VK как один из источников открытых публикаций. Любое событие желательно подтверждать через независимые источники: официальные страницы, СМИ, Telegram, MAX или сайт ведомства.
