# Чек-лист приватности и законности

Использовать перед любой учебной OSINT-задачей.

## 1. Цель

- [ ] Цель сформулирована ясно.
- [ ] Есть законная и этичная причина для анализа.
- [ ] Задача не направлена на вред, преследование или раскрытие частной информации.

## 2. Объект исследования

- [ ] Предпочтительный объект: организация, публичный объект, технический ресурс, новостное событие или синтетический учебный пример.
- [ ] Если затрагивается человек, используются только минимальные публичные сведения и только в безопасном контексте.
- [ ] Нет запроса на домашний адрес, телефон, документы, закрытые аккаунты или личную жизнь.

## 3. Источники

Разрешены:

- [ ] официальные сайты;
- [ ] публичные документы;
- [ ] открытые репозитории;
- [ ] публичные публикации;
- [ ] поисковые системы;
- [ ] открытые карты и справочники без нарушения приватности;
- [ ] синтетические датасеты и учебные примеры.

Не использовать:

- [ ] закрытые или сомнительные базы;
- [ ] сервисы, которые обещают частные сведения о людях;
- [ ] способы обхода доступа или приватности;
- [ ] данные, которые не нужны для цели.

## 4. Автомобильные номера

Разрешено:

- [ ] распознать номер на изображении;
- [ ] оценить качество распознавания;
- [ ] указать возможный формат, страну или регион по открытым правилам;
- [ ] использовать синтетические или учебные изображения;
- [ ] замазать номер в публичном отчёте.

Не делать:

- [ ] искать владельца автомобиля;
- [ ] искать адрес, телефон, профиль или место работы владельца;
- [ ] связывать номер с человеком;
- [ ] отслеживать маршруты или перемещения;
- [ ] использовать сомнительные сервисы поиска частных сведений.

## 5. Документирование

- [ ] Указаны источники и даты доступа.
- [ ] Факты отделены от предположений.
- [ ] Указан уровень уверенности.
- [ ] Указаны ограничения метода.
- [ ] Не сохранены лишние персональные данные.

## 6. Решение

- [ ] Продолжить задачу безопасно.
- [ ] Ограничить объём данных.
- [ ] Переформулировать задачу в учебную.
- [ ] Отказаться от небезопасной части.
