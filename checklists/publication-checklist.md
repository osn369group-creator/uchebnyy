# Publication Checklist

Перед сохранением, отправкой или публикацией материалов проверь:

## Минимизация

- [ ] Оставлены только данные, необходимые для задачи.
- [ ] Удалены лишние фрагменты изображения или документа.
- [ ] Удалены служебные метаданные, если они не нужны.

## Обезличивание

- [ ] Скрыты лица случайных людей.
- [ ] Скрыты адреса, телефоны, документы, QR-коды и штрихкоды.
- [ ] Скрыты уникальные идентификаторы, которые не требуются для отчета.
- [ ] Проверены отражения, фон, экранные уведомления и подписи файлов.

## Проверка вывода

- [ ] Факты отделены от предположений.
- [ ] Указаны источники и дата доступа.
- [ ] Указан уровень уверенности.
- [ ] Нет обвинительных формулировок без официального подтверждения.

## Финальное решение

- [ ] Материал можно безопасно сохранить.
- [ ] Материал можно безопасно передать адресату.
- [ ] Материал не содержит лишних частных сведений.
