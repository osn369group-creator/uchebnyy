# Расширенные безопасные упражнения для OSINT-агента

Все упражнения выполняются только на публичных, законных и безопасных объектах. Не использовать частных лиц, автомобильные номера для поиска владельцев, адреса, телефоны, маршруты, утечки или закрытые базы.

## Упражнение 1. Карта публичного сайта

Цель: научиться описывать структуру сайта организации.

Задание:

1. Выбрать публичный сайт организации.
2. Найти главные разделы.
3. Выписать публичные документы.
4. Определить, какие данные являются официальными.
5. Составить краткую карту сайта.

Результат:

```markdown
- Объект:
- Цель:
- Найденные разделы:
- Официальные источники:
- Наблюдения:
- Ограничения:
```

## Упражнение 2. Проверка новости

Цель: научиться проверять утверждение.

Задание:

1. Взять публичное утверждение из новости.
2. Найти первичный источник.
3. Найти независимый источник.
4. Сравнить формулировки.
5. Указать уровень уверенности.

## Упражнение 3. Анализ GitHub-репозитория

Цель: научиться извлекать знания из репозитория.

Задание:

1. Открыть публичный GitHub-репозиторий.
2. Прочитать README.
3. Определить назначение.
4. Найти основные файлы.
5. Описать, чему можно научиться.
6. Указать ограничения.

## Упражнение 4. OCR без нарушения приватности

Цель: научиться извлекать текст с изображения безопасно.

Задание:

1. Использовать изображение с учебным или публичным текстом.
2. Распознать текст.
3. Оценить качество распознавания.
4. Указать возможные ошибки.
5. Проверить, нет ли персональных данных.

## Упражнение 5. Безопасный анализ транспорта

Цель: научиться работать с изображением транспорта без поиска владельца.

Разрешено:

- описать видимые признаки;
- распознать номер как текст;
- оценить качество OCR;
- указать общий формат;
- предложить замазать номер перед публикацией.

Запрещено:

- искать владельца;
- искать адрес;
- искать телефон;
- искать маршрут;
- связывать автомобиль с частным лицом.

## Упражнение 6. Карточка доказательства

Цель: научиться фиксировать источник.

Задание:

1. Выбрать один источник.
2. Заполнить `templates/evidence-card.md`.
3. Отдельно указать, что источник подтверждает и чего не подтверждает.
4. Дать уровень уверенности.

## Упражнение 7. Обезличивание отчёта

Цель: научиться готовить безопасную публикацию.

Задание:

1. Взять учебный отчёт.
2. Найти чувствительные элементы.
3. Удалить или заменить их.
4. Сохранить фактическую полезность отчёта.

## Упражнение 8. Ограниченный вывод

Цель: научиться не преувеличивать результаты.

Задание:

1. Взять 3 слабых источника.
2. Составить вывод без категоричных обвинений.
3. Явно указать ограничения.
4. Предложить безопасные следующие шаги.
