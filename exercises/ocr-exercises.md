# Безопасные упражнения по OCR

Эти упражнения нужны для обучения распознаванию текста на изображениях без работы с частной информацией.

## Упражнение 1. Качество изображения

Возьми синтетическое изображение с коротким текстом.

Задача:

- определить видимый текст;
- описать качество изображения;
- указать факторы, которые могут привести к ошибке;
- дать уровень уверенности.

## Упражнение 2. Альтернативные варианты

Используй изображение, где часть символов похожа друг на друга: `O/0`, `I/1`, `B/8`, `S/5`.

Задача:

- дать основной вариант;
- дать альтернативные варианты;
- объяснить, почему возможна ошибка.

## Упражнение 3. Формат ответа

Оформи результат так:

```markdown
## Результат
Видимый текст: `...`
Альтернативы: `...`
Уверенность: низкая / средняя / высокая

## Основания
- качество изображения;
- видимые символы;
- возможные ошибки.

## Ограничения
- что не видно;
- что не подтверждено;
- какие выводы не делаются.
```

## Упражнение 4. Минимизация данных

Сделай учебный отчёт так, чтобы в нём осталась только техническая информация: качество изображения, результат OCR, уровень уверенности и ограничения.

## Правило

Использовать только синтетические, собственные или разрешённые изображения. Не добавлять в репозиторий реальные изображения с чувствительным контекстом.
