# Упражнения по проверке информации

## Упражнение 1. Надёжность источника

Выберите 3 публичных источника об одной теме.

Оцените каждый источник по критериям:

- кто автор или организация;
- дата публикации;
- есть ли первичные данные;
- есть ли ссылки на источники;
- есть ли признаки заинтересованности;
- можно ли подтвердить информацию независимо.

## Упражнение 2. Цепочка подтверждения

Выберите один факт из публичной новости.

Задача:

1. Найдите первоисточник.
2. Найдите вторичный источник.
3. Найдите независимое подтверждение.
4. Опишите, насколько факт подтверждён.

## Упражнение 3. Противоречия

Найдите два источника, которые описывают одно событие по-разному.

Задача:

1. Выпишите различия.
2. Найдите возможные причины различий.
3. Определите, какие утверждения подтверждаются лучше.
4. Укажите, где вывод должен оставаться осторожным.

## Упражнение 4. Уровень уверенности

Для каждого утверждения поставьте уровень уверенности:

- высокий;
- средний;
- низкий;
- не подтверждено.

Обязательно объясните причину оценки.
