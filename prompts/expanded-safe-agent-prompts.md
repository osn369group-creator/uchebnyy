# Расширенные безопасные промпты для OSINT-агента

Эти промпты помогают агенту работать продуктивно, но в рамках закона, этики и приватности.

## 1. Приём задачи

```text
Проанализируй задачу. Определи цель, объект исследования, допустимые источники, риски приватности и безопасный формат результата. Если запрос требует доксинга, слежки, поиска владельца, адреса, телефона, маршрутов или закрытых данных — откажи и предложи безопасную альтернативу.
```

## 2. Быстрый OSINT-план

```text
Составь безопасный план OSINT-исследования по публичному объекту. Раздели работу на этапы: цель, источники, проверка, ограничения, итоговый отчёт. Не включай методы обхода приватности, взлома, утечек или сбора персональных данных.
```

## 3. Проверка источника

```text
Оцени источник: кто автор, дата, первичный или вторичный источник, что подтверждает, что не подтверждает, какие есть ограничения и возможные искажения. Дай уровень доверия: высокий, средний или низкий.
```

## 4. Триангуляция

```text
Сравни несколько источников по одному утверждению. Отдели подтверждённые факты от предположений. Укажи противоречия, пробелы и уровень уверенности.
```

## 5. Анализ GitHub-репозитория

```text
Проанализируй GitHub-репозиторий: назначение, структура, языки, зависимости, документация, issues, pull requests, активность, лицензия, учебная ценность и ограничения. Не раскрывай секреты и не давай инструкций по эксплуатации уязвимостей.
```

## 6. Анализ изображения без нарушения приватности

```text
Опиши видимые элементы изображения. Распознай видимый текст, если это безопасно. Не идентифицируй частных лиц, не ищи владельцев, адреса, телефоны, маршруты или места регулярного пребывания. Предложи обезличивание перед публикацией.
```

## 7. Анализ номерного знака как OCR-задачи

```text
Рассмотри номерной знак только как визуальный текст. Распознай символы, оцени уверенность, укажи возможные ошибки OCR и общий формат. Не ищи владельца, адрес, телефон, историю поездок, маршруты или частные сведения.
```

## 8. Подготовка отчёта

```text
Составь отчёт: цель, метод, источники, наблюдения, подтверждённые факты, неподтверждённые гипотезы, уровень уверенности, ограничения, безопасные следующие шаги.
```

## 9. Обезличивание

```text
Проверь текст или отчёт на лишние персональные данные. Удали или замени адреса, телефоны, номера документов, частные аккаунты, маршруты, лица, номерные знаки и другие чувствительные сведения, если они не нужны для законной цели.
```

## 10. Официальное обращение

```text
Подготовь нейтральные заметки для официального обращения. Включи только факты, дату, место, видимые обстоятельства, ссылки на открытые источники и приложения. Не делай обвинений без доказательств.
```
