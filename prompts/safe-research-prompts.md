# Рабочие шаблоны для безопасного исследования

## Старт

```text
Сначала уточни цель, границы и разрешённые источники. Отделяй факты от предположений. Указывай уровень уверенности и ограничения.
```

## Проверка источника

```text
Оцени источник: автор, дата, независимое подтверждение, возможное устаревание, возможная предвзятость.
```

## Отчёт

```text
Собери краткий отчёт: цель, границы, найденные факты, источники, гипотезы, уровень уверенности, ограничения и следующие шаги.
```

## OCR изображения

```text
Определи только видимый текст на изображении. Оцени уверенность и ограничения. Не делай выводов, которые не следуют из изображения.
```

## Анализ изображения транспорта

```text
Опиши только видимые элементы изображения транспорта: тип, цвет, читаемые символы, качество кадра и ограничения. Не делай выводов о частных лицах.
```

## Подготовка нейтрального отчета

```text
Сформируй нейтральную заметку для законного обращения: что видно, какой контекст известен, что подтверждено, что не подтверждено, какие материалы приложить.
```

## Обезличивание

```text
Проверь материал перед публикацией. Перечисли элементы, которые нужно скрыть: лица, адреса, контакты, документы, коды, лишние идентификаторы и фоновые детали.
```

## Безопасная альтернатива

```text
Если задача выходит за рамки безопасного исследования, предложи учебный пример или общий разбор метода без частной информации.
```
