# Учебный план по ALPR/OCR

Цель: изучить технический пайплайн распознавания символов на изображениях и научиться документировать результат с уровнем уверенности.

## Модуль 1. Основы OCR

Изучить:

- что такое OCR;
- как качество изображения влияет на результат;
- почему модели ошибаются;
- как описывать уверенность.

Практика:

- выполнить упражнения из `exercises/ocr-exercises.md`;
- заполнить `templates/osint-report-template.md`.

## Модуль 2. Детекция области интереса

Изучить:

- bounding boxes;
- object detection;
- обрезку области интереса;
- влияние угла и разрешения.

Практика:

- сравнить ручное выделение области и автоматическое;
- описать ограничения.

## Модуль 3. OCR после детекции

Изучить:

- распознавание символов;
- похожие символы;
- confidence score;
- top-N варианты.

Практика:

- для каждого примера указать основной вариант и альтернативы;
- объяснить возможные ошибки.

## Модуль 4. Форматы и валидация

Изучить:

- проверку структуры текста по открытым правилам;
- почему формат не является подтверждением личности или факта владения;
- как избегать чрезмерных выводов.

Практика:

- проверять только структуру;
- не делать выводов, которые не подтверждены источниками.

## Модуль 5. Приватность

Изучить:

- минимизацию данных;
- замазывание чувствительных фрагментов;
- хранение только учебных и синтетических примеров;
- отчёт без лишнего контекста.

## Репозитории для технического изучения

- `openalpr/openalpr` — классический ALPR-пайплайн.
- `ankandrew/fast-alpr` — современный Python-подход.
- `ankandrew/fast-plate-ocr` — OCR-компонент.
- `parkpow/deep-license-plate-recognition` — API и JSON-результаты.
- `Muhammad-Zeerak-Khan/Automatic-License-Plate-Recognition-using-YOLOv8` — учебный YOLOv8-пайплайн.
- `MicrocontrollersAndMore/OpenCV_3_License_Plate_Recognition_Python` — классический OpenCV-подход.

## Итоговый навык агента

Агент должен уметь:

- объяснить технический пайплайн;
- оценить качество изображения;
- дать результат OCR с осторожной формулировкой;
- указать альтернативные варианты;
- описать ограничения;
- не делать выводов о людях или частной информации.
