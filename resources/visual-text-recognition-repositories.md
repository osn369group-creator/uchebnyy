# Public Visual Text Recognition Repositories

Цель: изучать технический pipeline распознавания текста на изображениях.

## Использовать для

- Поиск области с текстом.
- OCR распознавание символов.
- Оценка уверенности.
- Анализ ошибок распознавания.
- Обезличивание результатов перед публикацией.

## Учебные репозитории

| Репозиторий | Направление |
|---|---|
| `parkpow/deep-license-plate-recognition` | API-подход и JSON-ответы |
| `kby-ai/Automatic-License-Plate-Recognition-Flutter` | мобильный пример Flutter |
| `kby-ai/Automatic-License-Plate-Recognition-Android` | мобильный пример Android |
| `afeefabubakar/License-Plate-Detection-with-OpenCV` | OpenCV-подход |
| `LdDl/license_plate_recognition` | классический pipeline распознавания |
| `gyupro/EasyKoreanLpDetector` | региональный пример детекции |

## План изучения

1. Прочитать README каждого проекта.
2. Зафиксировать входные данные и выходной формат.
3. Выписать этапы: поиск области, обрезка, OCR, post-processing.
4. Проверить требования к качеству изображения.
5. Сравнить типичные ошибки распознавания.
6. Описать ограничения.
7. Подготовить учебный вывод без частных данных.
