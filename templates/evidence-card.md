# Шаблон карточки доказательства

Используется для фиксации одного наблюдения или источника в OSINT-исследовании.

```markdown
# Evidence Card

## ID

EVID-0001

## Краткое описание

...

## Источник

- Название:
- URL или путь к файлу:
- Тип источника: официальный / независимый / вторичный / слабый сигнал
- Дата публикации:
- Дата доступа:

## Что подтверждает

...

## Что НЕ подтверждает

...

## Цитата или краткая выдержка

...

## Контекст

...

## Возможные ошибки интерпретации

...

## Уровень уверенности

Высокий / средний / низкий

## Проверка

- [ ] Проверено по первичному источнику
- [ ] Проверено по независимому источнику
- [ ] Проверена дата
- [ ] Проверен контекст
- [ ] Не содержит лишних персональных данных

## Решение

Использовать / не использовать / требуется дополнительная проверка
```

## Правила

- Не включать лишние персональные данные.
- Не сохранять адреса, телефоны, маршруты или частные сведения без законной необходимости и согласия.
- Не использовать утечки и закрытые базы.
- Отделять наблюдение от вывода.
