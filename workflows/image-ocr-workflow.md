# Image OCR Workflow

Цель: извлечь видимый текст с изображения и оформить проверяемый результат.

## 1. Подготовка

- Сохранить исходный файл отдельно.
- Сделать рабочую копию.
- Зафиксировать дату, источник и контекст получения файла.

## 2. Первичный осмотр

- Оценить качество изображения.
- Отметить, какие области содержат текст.
- Не делать выводы о людях или событиях без подтверждения.

## 3. OCR

- Распознать видимый текст.
- Зафиксировать результат как черновик.
- Отметить сомнительные символы знаком `?`.
- Указать уровень уверенности.

## 4. Проверка

- Сравнить результат с исходным изображением.
- Проверить возможные ошибки: O/0, I/1, B/8, S/5, Z/2.
- При низком качестве указать, что результат ненадежен.

## 5. Обезличивание

- Перед публикацией скрыть все лишние идентификаторы.
- Не сохранять больше фрагментов, чем нужно для отчета.

## 6. Отчет

- Исходный вопрос.
- Метод.
- Результат OCR.
- Сомнительные места.
- Уровень уверенности.
- Ограничения.
