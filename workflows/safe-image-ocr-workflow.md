# Безопасный рабочий процесс: OCR изображения

Этот процесс предназначен для учебного распознавания текста на изображениях и аккуратного документирования результата.

## Разрешённая цель

- определить, какой текст виден на изображении;
- оценить уверенность распознавания;
- объяснить качество изображения и ограничения;
- использовать учебные, синтетические или разрешённые примеры.

## Перед началом

Проверить:

- [ ] изображение можно анализировать;
- [ ] цель учебная, техническая или защитная;
- [ ] нужен только видимый текст;
- [ ] результат не связывается с частной информацией.

## OCR-пайплайн

1. Найти область с текстом.
2. Обрезать область интереса.
3. Улучшить читаемость: масштаб, контраст, шум, наклон.
4. Выполнить OCR.
5. Нормализовать символы.
6. Проверить возможный формат по открытым правилам.
7. Указать уровень уверенности.

## Оценка качества

Факторы, снижающие уверенность:

- размытие;
- блики;
- сильный угол;
- низкое разрешение;
- закрытые символы;
- нестандартный шрифт;
- грязь или повреждение объекта.

## Формат краткого ответа

```markdown
## Результат
Предполагаемый текст: `...`
Альтернативные варианты: `...`
Уверенность: низкая / средняя / высокая

## Почему так
- видимые символы;
- качество изображения;
- возможный формат.

## Ограничения
- возможна ошибка OCR;
- дополнительные выводы не делаются.

## Приватность
Анализ ограничен распознаванием видимого текста и не связывается с частной информацией.
```

## Правило хранения

В публичном репозитории хранить только учебные заметки, синтетические примеры и общие методы. Реальные изображения и контекст, позволяющий идентифицировать людей, не сохранять.
