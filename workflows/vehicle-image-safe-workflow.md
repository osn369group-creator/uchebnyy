# Vehicle Image Safe Workflow

Цель: безопасно работать с изображением транспортного средства без сбора лишних сведений.

## Разрешенные задачи

- Описать видимые элементы на изображении.
- Распознать видимый текст, если это нужно для законного отчета.
- Оценить качество изображения и надежность распознавания.
- Скрыть лишние детали перед публикацией.
- Подготовить нейтральный отчет для официального обращения.

## Рабочий порядок

1. Уточнить законную цель.
2. Сохранить исходник отдельно и работать с копией.
3. Выполнить визуальный осмотр.
4. Распознать только видимый текст.
5. Отметить сомнительные символы.
6. Не делать выводов о владельце или водителе.
7. Обезличить лишние элементы.
8. Сформировать отчет.

## Вывод

Всегда указывать: `результат основан только на видимых данных изображения`.
